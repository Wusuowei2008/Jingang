## 一、设计系统总纲

### 品牌定位
**核心关键词**：探索感、知识感、温暖、可信赖
**目标用户**：家长、老师、研学机构
**设计语言**：Editorial（杂志感）× Warm Academic（暖色学院风）

### 色彩系统

```css
:root {
  /* 主色：深林绿，传递知识与自然探索感 */
  --color-primary:     #2D5016;
  --color-primary-mid: #4A7C2F;
  --color-primary-light: #E8F0E0;

  /* 辅色：暖土黄，传递温暖与人文气息 */
  --color-accent:      #C4852A;
  --color-accent-light: #F5E6C8;

  /* 中性色 */
  --color-ink:         #1A1A18;
  --color-ink-muted:   #5C5C54;
  --color-paper:       #FAF8F3;   /* 米白背景，非纯白 */
  --color-paper-dark:  #F0EDE4;

  /* 功能色 */
  --color-success:     #3A7D44;
  --color-warning:     #D4792A;
  --color-error:       #C0392B;

  /* 排版 */
  --font-display: 'Noto Serif SC', Georgia, serif;   /* 标题：衬线，有文化感 */
  --font-body:    'Noto Sans SC', system-ui, sans-serif;  /* 正文：清晰易读 */
  --font-mono:    'JetBrains Mono', monospace;

  /* 间距 */
  --radius-sm:  6px;
  --radius-md:  12px;
  --radius-lg:  20px;
  --radius-xl:  32px;
}
```
